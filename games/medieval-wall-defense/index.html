Reduced Motion

Always Show Hero Ranges
Draw every hero's range circle on the field, not just the one you've clicked. Useful on mobile where the slot panel covers the heroes.
Auto-Dismiss Events
Automatically close mid-run event popups after a delay if you don't pick an option. Designed for idle or overnight play. Dismissing gives no reward and no penalty — clicking any option cancels the timer and resolves the event normally.
AFK Auto-Restart
If a wave takes longer than the delay to clear, end the current run (banking any earned RP) and start a new one. Combined with commanders and Auto-Dismiss Events, lets you leave the game running overnight to farm RP. No cap on restart frequency.
Auto-Buy Research (RP)
Once per second, spend RP on the cheapest affordable repeatable research node (Sharper Stakes, Hotter Coals, Keen Arrows, etc.). One-shot nodes always require a manual click. Crown-cost nodes never spend Crowns — that's a separate toggle. Designed for overnight farmers who bank RP across many AFK runs.
Auto-Spend Crowns
Once per second, spend Crowns on the cheapest affordable repeatable crown-cost node. Crowns are slow to accumulate, so this rarely fires — but it closes the loop for unattended progression. Off by default since most crown-cost nodes are skin unlocks (one-shot, never auto-bought) or keystones (one-shot).
Auto-Prestige
After your first manual prestige, automatically prestige on every gameover where banked RP meets the threshold below. Re-uses the most-recent ancestry choice and shard preset. The first prestige always requires manual confirmation so you can pick your starting ancestry.
Restart Tutorial

End Run · Claim RP
End the current run now and bank your earned Research Points. You'll see the wall-breach screen, where you can spend RP and start fresh.
Reset Current Run
Clear enemies, coins, and heroes and start over without claiming any RP. Research is kept.
Wipe Everything
Full factory reset — erases all research, RP, Crowns, prestige progress, commanders, achievements, stats, and daily scores. You'll be sent back to the main menu and the tutorial will replay. Cannot be undone.
Export Save
