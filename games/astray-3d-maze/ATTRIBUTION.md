## Description

A WebGL-based 3D first-person maze game built with Three.js and Box2dWeb physics engine. The player controls a ball through a randomly-generated maze using keyboard and mouse.

## Technology

- **Engine:** Three.js (WebGL rendering)
- **Physics:** Box2dWeb (2D rigid body dynamics)
- **Controls:** Keyboard + Mouse/Tilt
- **Save:** localStorage (browser-local game state)
