Text rendering (2026-07-08 adaptation): the upstream repo bundled
vector_battle_regular.typeface.js, a canvas vector-font conversion of the
"Vector Battle" font (c) 1999 by "ck!" / freakyfonts, which carries only a
bare copyright notice and empty license fields - no permissive license
grant. That file is NOT vendored here. game.js's Text.renderText was
rewritten to draw with the browser's built-in canvas font/strokeText API
instead (no font file, no attribution needed - system fonts are not
redistributed).

Rebrand note: the upstream project's README and repo name use the literal
title "Asteroids", which is a registered Atari trademark for the 1979
arcade game. This vendored build is retitled "Asteroid Blaster" - a
generic, descriptive name - and no Atari branding, sprites, or audio are
used; the game mechanic (rotate/thrust ship, shoot rocks that split into
smaller rocks, screen-wrap) is a public-domain-style arcade archetype
implemented from scratch in this MIT-licensed codebase.
